# Ensolvers ToDo List

- This project was created using **Spring Boot 2.5.6**

**Dependencies**
- Apache Maven 3.6.3
- JDK 11
- Kotlin 1.5.31
- PostgreSQL 14.0
- Nodejs 10.19.0
- NPM 6.14.4

**HOW TO RUN**
- Setup a new postgresql server "todo-list"
- Execute command `bash init.sh` from main directory
- In another terminal window execute `bash init-react.sh`
